개발자로서 성장하기 위한 지식들을 정리하고 공유하는 프로젝트입니다.
프로젝트는 계속해서 제가 모르는 지식들을 정리한다는 느낌으로 다이어그램을 그리고, 원리를 작성합니다.
작성된 글들을 하나의 메모지로서 활용하여 제가 배웠던 기술들을 다시 복습을 할 수 있음과 동시에 다른 개발자들에게도 해당 기술에 대해서 한눈에 파악할 수 있는 기회가 되었으면 좋겠습니다.

기술들의 분리는 커다란 덩어리로 구성되어 있으며, 하나의 유기체라고 생각되는 컴퓨터의 프로그래밍 세계의 구성요소들이 머릿속에 한번에 그려질 수 있도록하는 것이 프로젝트의 최종 목표입니다.
